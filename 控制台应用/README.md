\# houboyi-ConsoleApplication



一个简单的 C++ 控制台应用程序，用于查询字符的 Unicode 编码。



\## 版本

\*\*当前版本\*\*：`Version 0.1 Alpha`  

\*\*状态\*\*：\*\*内部测试版\*\*，功能不全，已知 bug。



\## 功能

\- 查询字符的 Unicode 值（当前仅支持无空格字符串）



\## 已知问题

\- 输入字符串时，空格会导致截断

\- 输入非数字选项时，程序可能崩溃

\- 功能1仅为占位符，尚未实现
