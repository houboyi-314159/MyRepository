## 贡献指南
欢迎提交 Issue 和 Pull Request！但请注意：

### 提交前请确认
- 你的代码**符合项目风格**（目前是简单的C++控制台程序）。
- 你的修改**是必要的**（修复bug、改进性能、增加重要功能）。
- 你**已经测试过**代码，确保不引入新bug。

### 提交规范
1. **Fork 本仓库**，并在你的仓库中创建分支。
2. **提交信息**请用英文或清晰的中文描述修改内容。
3. **一个PR只做一件事**（比如修复一个bug或添加一个功能）。
4. 如果可能，**添加测试**或更新文档。

### 注意事项
- 我**不保证**会合并所有PR（时间有限，且项目主要是个人学习用）。
- 如果你的PR涉及**重大架构修改**，请先开Issue讨论。
- 提交PR即表示你**同意**你的代码按本项目许可证（MIT）授权。

感谢你的贡献！
